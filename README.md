# Mensaje Binario

Este proyecto presenta un mensaje en código binario que se desplaza hacia arriba en una animación en una página web. El mensaje es un texto en binario que, cuando se interpreta, forma una frase legible.

## Características

- Animación en la que el mensaje binario se desplaza de abajo hacia arriba.
- Diseño minimalista con fondo negro y texto en color verde, emulando un estilo retro de consola o terminal.
- El mensaje está codificado en binario y se muestra en un formato legible para las personas familiarizadas con este sistema.
